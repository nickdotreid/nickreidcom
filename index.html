    Sample Work

    DSRIP

    I created a web application for the California Hospitals Association which is now being used by about 20 public hospitals in the state of California. This web application streamlines the state manditated incentive funding by translating an excel based document. I built the website using Django and Backbone.js.

    San Francisco HIV Charts

    I worked with epidemiologists to experiment with HIV epidemiology data while working at the HIV Prevention Section of the San Francisco Department of Public Health. I designed and developed a social chart analysis and annotation tool, which aimed to create create a real time social environment for data. I presented this project at American Public Health Association in 2012.
